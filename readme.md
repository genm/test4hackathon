# Smart Contract for Neo Hackathon


# 実行
```
npm run start
```
でmain.jsが実行されます。
これは
```
babel-node main.js
```
を実行しているのと同義です。


その他のスクリプトについては
```
npm run
```
で実行できるものを探してください

package.jsonのscripts:の中に自分が使いやすいように

# 開発に関して

## パッケージの追加
```
npm i hogehoge@hoge --save-dev
```
